المميزات الأسئلة تواصل
مخصص للكافيهات والمطاعم الصغيرة — يعمل Offline

برنامج كاشير بسيط وسريع — بدون إنترنت

بيع سريع، خصم خامات تلقائي، مصروفات ومشتريات، تقارير يومية، وإيراد (المبيعات - المصروفات) — كل شيء بالعربي.

اطلب تجربة / عرض سعر شوف المميزات
المكان: الإسكندرية / مصر
التشغيل: Windows + SQLite
الدعم: تركيب + تدريب + صيانة
لقطة سريعة
  • نقطة بيع (POS) سريعة
  • مخزون خامات + تنبيهات نقص
  • مصروفات + مشتريات مرتبطة بخامة
  • ملخص اليوم + تصدير Excel (CSV)
  • حضور وانصراف + تقرير أسبوعي

المميزات

المبيعات
  • نقطة بيع سهلة وسريعة
  • خصم وملاحظات + بقشيش
  • عرض الطلبات والتفاصيل
المخزون والخامات
  • إدارة خامات وحدّ تنبيه
  • خصم خامات تلقائياً عند البيع
  • سجل حركات المخزون
المصروفات والمشتريات
  • تسجيل مصروفات يومية
  • مشتريات مرتبطة بخامة ترفع المخزون
  • تقارير الإيراد
التقارير
  • ملخص اليوم
  • تصدير Excel (CSV)
  • تقرير الإيراد (مبيعات - مصروفات)

أسئلة شائعة

هل البرنامج يحتاج إنترنت؟
لا، يعمل بالكامل Offline على نفس الجهاز.
هل يدعم طباعة فواتير؟
يمكن ربطه بالطابعة حسب نوع الطابعة (يتم تحديدها أثناء التركيب).
